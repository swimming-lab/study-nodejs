# study-nodejs

인프런 강의 및 기타 학습 채널을 통한 nodejs 학습

### 실행
```
node hello.js
node webserver.js
node underscore.js
```

### 의존성
```
npm install underscore --save
npm install uglify-js -g
```

### uglifyjs 실행
```
uglifyjs [input files] [options]
uglifyjs ./uglify/test.js
```
